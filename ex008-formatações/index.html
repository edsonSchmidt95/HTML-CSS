principais formatações

negrito/ destaque

nessa frase temos um termo em negrito usando a tag B (não semantica)

nesta frase tenho um termo em destaque usando a tag strong

italico / enfase

nesta frase temos um termo em italicado usando a tag I, não semantico

nesta frase temos um termo em enfase usando a tag EM

texto marcado

podemos criar também um texto marcado usando a tag MARK, e posso mudar a cor usando a tag style dentro da tag mark e escolher a cor

texto grande e pequeno

estamos criando um texto grande e um texto pequeno neste paragráfo

texto deletado

podemos marcar um texto excluido para indicar que ele deve ser lido mas não considerado

texto inserido

podemos marcar um texto como inserido para dar um enfase e indicar que ele foi adcionado depois

texto sobrescrito

para inserir coisas do tipo x20+3 , para deixar o 20 no alto, se usa a tag SUP

texto subscrito

para inserir coisas do tipo h2o ,para deixar o 2 em baixo, se usa a tag SUB
