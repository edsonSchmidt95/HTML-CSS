HTML básico

perceba que o h1 "criando site com html e css" está em outra formatação que os demais h1, isso por que ele está identificado como h1 id="principal" dentro do códig html, assim quanso quisermos editr ou modificar algo em css só nele, usamos a tag h1#principal, e não se pode ter dois ou mais ID na mesma documentação, quando se quer mudar ais de um e usa a tag class. class pode ter varios e com varios nomes, mas ID é somente um por documento. como diria guanabara, funciona mas não ta certo

HTML intermediario

já no css, para identificar um clas se usa um ., exempo .intermediario assim tudo que for posto vai mudara em todos os que estiver com essa identificação

HTML avançado

id no css vira #, só pode um elemento
e class no css vira . pode varios elementos
